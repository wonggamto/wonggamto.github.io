在github上创建博客

github上搭建自己的blog 安装 由于我用的是macos，所以先介绍mac的安装方式： 终端输入下面两条命令: brew install hugo 这个时候会进入安装 hugo version 如果上面的命令能返回一个版本号，则表示安装成功 配置 打开hugo的官网,然后点击Quick Start Quick Start会自动识别你的系统然后给出对应的操作提示 终端进入 github.io-generator目录,输入 hugo new posts/my-first-post.md 其中my-first-post指的是你的博客文章的名字，例如: 在vscode的左侧边栏可以看到这时候博客已经创建成功了，如图: 仔细观察，draft的参数是true，代表当前是草稿模式，就是说发布以后别人是看不见的，需要在文章写完以后改成false 写完内容以后，输入命令: hugo server -D 点击链接，即可打开刚才写好的文章了。 打开config.toml 文件可以修改一些对应的配置,zh-Hans是简体中文的意思 创建一个 .gitignore 文件 填写如图所示配置，让public自成一个仓库 进入public目录 输入 git init git add . git commit -m “git创建博客教程”
